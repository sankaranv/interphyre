# Level Viewer Example

Explore and visualize PHYRE levels to understand the physics puzzles.

## Overview

View individual levels or batch process multiple levels with solutions.

**Complexity:** Beginner
**Runtime:** Depends on mode

## Usage

```bash
# View single level
python demos/level_viewer.py

# Batch mode (visualize solutions)
python demos/level_viewer.py --batch
```

## What It Shows

- Level geometry and objects
- Action space (where balls can be placed)
- Success/failure outcomes
- Solution trajectories (in batch mode)

## Use Cases

- Understanding PHYRE level structure
- Exploring available puzzles
- Visualizing solutions
- Initial exploration of the environment
